# 야니의 소설창고 v5

여러 소설을 회차 단위로 기획·집필·복사하는 라이트모드 개인 소설 집필 워크스페이스입니다.

## v5 변경점

- 앱 이름을 `야니의 소설창고`로 변경
- 펼친 책 + 작은 펜촉을 결합한 심플한 브랜드 아이콘 적용
- 브라우저 탭 파비콘에도 동일한 아이콘 적용
- 책 표지 설정 모달 추가
- 단색 표지 프리셋 10개 제공
- 사용자가 직접 커스텀 컬러 선택 가능
- 표지 이미지 업로드 지원
- 업로드 이미지를 지우면 선택한 단색 표지로 복귀
- 기존 이미지 업로드 표지와 저장 데이터 유지
- 기존 회차 부분 복사 4종, 작품 전체 복사 4종, 작품 기획 개별/전체 복사 유지

## 실행

`index.html` 또는 `야니의_소설창고_v5_단일파일.html`을 브라우저에서 엽니다.

브라우저 로컬 저장 방식을 사용하므로, 실제 장기 사용 서비스로 전환할 때는 Supabase 등 영구 저장소 연결을 권장합니다.
